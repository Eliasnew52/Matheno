Proyecto Final de CS50: Matheno


¿Qué es Matheno?

Matheno es una plataforma web que permite a los usuarios crear y participar en quizzes personalizables. Ofrece opciones para editar el formato del test, agregar multimedia y definir métodos de respuesta. Los usuarios pueden filtrar y buscar tests en la página principal. Cada usuario tiene un perfil obligatorio para seguir a otros, recibir notificaciones y ver estadísticas comparativas después de completar un test.
